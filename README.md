# AgyntiQ

Premium enterprise AI company website built with Next.js, React, TypeScript, Tailwind CSS, Framer Motion, GSAP, and Lenis.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Brand

- Logo: `main-logo.png`
- Primary colors: blue, purple, white, deep black, dark navy

## Notes

- The site is optimized for a flagship enterprise AI look and feel.
- The current landing page is in `src/components/aurora-landing.tsx`.
