## 🛠 Technical Implementation

### File Structure

```
src/
├── app/
│   ├── layout.tsx              # Root layout with SEO metadata
│   ├── page.tsx                # Main page component
│   ├── globals.css             # Global premium styles
├── components/
│   ├── agyntiq-website.tsx     # Main website (1500+ lines)
│   ├── ui-components.tsx       # Reusable components
│   ├── advanced-animations.tsx # Animation utilities
│   ├── aurora-landing.tsx      # Legacy component
```

### Key Technologies

| Tech          | Purpose                    |
| ------------- | -------------------------- |
| React 19      | UI components              |
| Next.js 15    | Framework & optimization   |
| TypeScript    | Type safety                |
| Tailwind CSS  | Styling & design system    |
| Framer Motion | Component animations       |
| GSAP          | Advanced scroll animations |
| Lenis         | Smooth scroll experience   |

- ✅ Build time: ~17.7s
- ✅ Page size: ~99.3 KB (route) + ~102 KB (shared)

- ✅ Static generation for fast loading
